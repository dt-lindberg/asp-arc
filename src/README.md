# Baseline Setup
This setup intends to be a baseline for solving ARC-AGI puzzles with Qwen-3.6-35B-A3.

```
* agent/          stores LLM and harness related files  .
* arc-puzzles/    contains ARC-AGI-1 and 2 repo clones  .
* config/         defines all config values             .
* prompts/        all prompt-templates                  .
* utils/          general collection of helper code     .
```
